## About The Project
📌 ATM Interface (Java Project):

This project is a Java-based ATM Interface developed using Object-Oriented Programming (OOP) principles. It simulates basic ATM operations such as checking balance, depositing money, and withdrawing funds.
The application runs in the console and provides a simple menu-driven interface for user interaction.

## 🎯 Objectives

Implement OOP concepts like classes, objects, encapsulation, and methods
Simulate real-world ATM operations
Handle user input and validate transactions


## ⚙️ Features
- ✅ Check Account Balance
- ✅ Deposit Money
- ✅ Withdraw Money
- ✅ Input Validation
- ✅ Menu-driven Interface
- ✅ Error Handling (invalid input, insufficient balance)

## 💻 Technologies Used

- Java (JDK 8 or above)
- VS Code (recommended IDE)
